Citation du Jour

Citation du Jour est une application mobile que j'ai développée pour offrir aux utilisateurs une nouvelle citation inspirante chaque jour. L'application s'appuie sur une base de données personnalisée et une API hébergée sur Heroku pour récupérer les citations.

Cette application est un excellent exemple de mon travail avec les technologies suivantes :

  • Swift pour le développement iOS
  • Node.js et Express pour l'API back-end
  • Base de données MongoDB pour stocker les citations
  • Heroku pour l'hébergement de l'API

Vous pouvez télécharger l'application depuis l'App Store et explorer le code source et plus de détails sur GitHub.
